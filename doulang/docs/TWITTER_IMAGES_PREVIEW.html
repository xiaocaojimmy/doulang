How to Use These Designs

  1. Screenshot: Press F12 to open dev tools, then Ctrl+Shift+P > "Capture full size screenshot" (Chrome)
  2. Print to PDF: Ctrl+P, select "Save as PDF", set paper size to custom 1200x675
  3. Canva: Use these as reference to recreate in Canva with exact colors and layout
  4. Figma: Import this HTML or use the design specs in IMAGE_DESIGNS.json

Color Palette: Primary #667eea, Secondary #764ba2, Success #51cf66, Error #ff6b6b

Font: System fonts (-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto)

Twitter Dimensions: 1200x675 pixels for optimal display
